HTML and CSS Documentation

HTML Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Inline HTML Tags

Tag Name Description
<a> Anchor Creates a hyperlink to other web pages or resources.
<img> Image Embeds an image in the HTML document
<span> Span Defines an inline container used to mark up parts of text for styling
<abbr> Abbreviation Defines an abbreviation or an acronym.
<br> Line Break Inserts a single line break
<code> Code Defines a piece of computer code.

CSS Selectors

Selector Name Example Description Specificity
* Universal Selector * {} Selects all elements on the page. Very Low (0,0,0,0)
element Type Selector div Selects all specific HTML tags (like div, p, h1). Low (0,0,0,1)
.class Class Selector .box Selects elements with a specific class. Medium (0,0,1,0)
#id ID Selector #header Selects an element with a specific ID. High (0,1,0,0)
[attr=value] Attribute Selector [type="text"] Selects elements with a matching attribute and value. Medium (0,0,1,0)
element.class Combined Selector div.box Selects all <div> elements with class box. Medium-High (0,0,1,1)
parent > child Child Combinator ul > li Selects direct children of an element. Depends (based on parts)
element element Descendant Combinator div p Selects all nested elements (any level deep inside the parent). Depends (based on parts)
a:hover Pseudo-class Selector a:hover Applies styles when a link is hovered. Medium (0,0,1,1)
p::before Pseudo-element Selector p::before Inserts content before the content of the <p> element. Low (0,0,0,2)

CSS Box Model & Layout Properties

Property Name Example Description Value Type
margin Margin margin: 20px; Sets space outside the element's border. Length / Auto / %
padding Padding padding: 15px; Sets space between content and border. Length / %
border Border border: 1px solid black; Sets the border around an element. Shorthand (width style color)
width Width width: 100%; Defines the width of the element. Length / % / Auto
height Height height: 200px; Defines the height of the element. Length / % / Auto
display Display Mode display: flex; Controls layout behavior of an element. block / inline / flex / grid / none
position Positioning position: absolute; Defines how an element is positioned in the document. static / relative / absolute / fixed / sticky
top Offset - Top top: 10px; Moves the element down from the top edge (used with position). Length / % / Auto
z-index Z-Index z-index: 10; Controls the stack order of overlapping elements. Integer
overflow Overflow overflow: hidden; Controls what happens to content that overflows the element's box. visible / hidden / scroll / auto
